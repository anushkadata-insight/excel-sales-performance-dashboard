# Excel Sales Performance Dashboard

## Project Overview
This project analyzes sales performance using Excel with pivot tables and interactive charts.

The dataset contains 700+ salespersons selling different products across multiple geographies including India, Australia, and Canada.

## Dataset Features
- Salesperson name
- Geography
- Product
- Number of customers
- Sales amount

## Dashboard Sheets

### Pivot Table 1 – Salesperson Analysis
This pivot table allows selection of a specific salesperson.  
Based on the selected salesperson, the dashboard dynamically updates:

- Products sold
- Total sales amount

This helps analyze individual salesperson performance.

### Pivot Table 2 – Geography Analysis
This sheet allows users to select a geography such as:

- India
- Australia
- UK
- Canada

The pivot table then displays:
- Salesperson names
- Total sales amount within that geography

This helps analyze regional sales performance.

### Pivot Chart – Salesperson Comparison
A bar chart visualizes the total sales of different salespersons.

This makes it easier to quickly identify top performers.

### Percentage Report
The final sheet contains:

- Salesperson name
- Sales amount in USD
- Sales amount converted to INR
- Percentage contribution chart

This shows each salesperson's contribution to overall revenue.
